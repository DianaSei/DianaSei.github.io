TIt's an easy and addictive game which develops your quick-thinking and fast response. Designing a game which includes a fast reply from the user makes your work a bit challenging as well. As the player might not notice some of the details of the design focusing only on the main area (i.e. here it's an up area where a ball drops and a wheel which should be matched with the same color of the dropping ball). Therefore, the pause button was confirmed to be inside of the wheel as most of the users will tap to spin the circle on the right/left sides of the wheel (usually a right side of the wheel) that makes it not only a playing tool but a control button as well. Such indicators like level and score are shown on the top where the main focus is. This makes easier for a user to monitor the game and his/her score.
